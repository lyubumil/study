WEB

  • web과 인터넷의 차이점은 무엇일까?
    • 인터넷은 전세계를 연결하는 국제 정보 통신망이다. (해저케이블로 전세계가 연결이 되어있다.)
    web은 인터넷을 통해 작동하는 서비스 이다.(사람들이 정보를 공유하는곳)

  • 인터넷의 역사
    • 1960년 최조의 인터넷 ARPANET
    1974년 TCP/IP개발
    1990년 유럽입자물리연구소 팀 버너스리 'web'발명(HTML) -> 문자 주고받기만 가능했다
    1993년 최조의 웹브라우저 모자익 개발
    1994년도부터 우리가 알고있는 익스플로어나 크롬 같은 웹브라우저가 만들어지기 시작 (웹브라우저 전쟁)

  • web client 와 web server
    • web client는 실제 사용자 => 클릭했을 때 클릭한 페이지를 제공하는 pc
    web server는 데이터를 돌려주는 사랍 ex)네이버, 다음 => 클릭한 페이지를 제공하는 pc
    web client가 주소를 입력해 요청을 하면 web server에서 코드를 넘겨주며 웹브라우저에서 보이게 된다.
    f12를 누르면 개발자도구가 나오는데 이러한 코드들을 web server에서 web client로 넘겨주는 것이다.

  • 프로토콜 Protocol
    • 프로토콜은 컴퓨터와 컴퓨터간 원활한 통신을 위해 지키기로 한 약속(규약)이다.

  • HTTP
    • Hyper Text Transfer Protocol 의 약자로 하이퍼텍스트를 기반으로 데이터를 전송하는 프로토콜이다.
    인터넷에서 하이퍼텍스트 문서를 교환하기 위하여 사용되는 통신규약이다.

    홈페이지는 크게 HTML(뼈대) + CSS(디자인) + JS(기능)로 만들어졌다
